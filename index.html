How to Use
1. Pick a distro below, or paste a custom URL
2. Click Start — images download from the web
3. Booting takes 30 s – 3 min depending on distro
4. Click the black screen to capture keyboard
5. Press Escape to release keyboard focus

Custom images must be served with
Access-Control-Allow-Origin: * headers.

Tip: Open Terminal Log in the toolbar to
see serial output while booting.
